# Python-Project

This Python project presents a versatile temperature converter calculator designed to effortlessly convert temperatures between Celsius, Fahrenheit, and Kelvin scales. With a streamlined command-line interface, users can swiftly input temperature values in any of the supported scales and obtain accurate conversions across various temperature units.

Supported Conversions:
1. Celcius to Fahrenheit, vice versa
2. Celcius to Kelvin, vice versa
3. Fahrenheit to Kelvin, vice versa
